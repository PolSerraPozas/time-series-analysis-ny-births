# Time Series Analysis in R: NYC Births Dataset

## Objective
Analyze a time series dataset of monthly births in New York City and model its trend and seasonal components.

## Dataset
Monthly birth counts (1947–1959).

## Source
http://robjhyndman.com/tsdldata/data/nybirths.dat

## Methods
- Linear regression to estimate trend
- Seasonal decomposition (additive model)
- Calculation of seasonal coefficients
- Time series reconstruction
- Prediction of future values

## Tools
- R (base functions)

## Key Steps
1. Load and visualize time series
2. Estimate linear trend
3. Remove trend component
4. Compute seasonal coefficients
5. Build additive model
6. Forecast future values

## Results
- Identified an increasing linear trend
- Detected clear seasonal patterns
- Built a model capable of forecasting future birth counts
